About this project

This project is intended to explore the catechism in a digital way. I plan on using tei to analyze the text. The tei catechism was inspired by both Fr. Mike Schmitz catechism in a year and St. Carlo Acutis. This project is independent of any university or organization.

About me

My name is Samantha Moniot and I am a digital media student at Penn State Behrend. I was raised Catholic and grew up attending Immaculate Conception in Monrovia,Ca. During College I have learned a lot about the Catholic Church and digital humanities which also inspired me to start this project.
